
# School.app

A School management system made to be quick and intuative to use.
## Functions

- Manage Student

- Manage Teacher

- Manage Modules

- Manage Enrollment

- Manage Student Grades


## Features

- Robust Error free experience
- Password Hashing using SHA-512 algorithim 


## Roadmap
